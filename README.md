# Skyline-Metric
Skyline Metric ist ein Wetter-Dashboard im Glow-Glasmorphismus-Design. Es bietet Echtzeitdaten, interaktive ApexCharts für 24h/30-Tage-Trends sowie einen Stadt-Vergleichsmodus. Mit Regen-Vorschau, Favoritenliste und Skeleton-Loadern vereint es Ästhetik mit funktionaler Tiefe für Desktop und Mobile
